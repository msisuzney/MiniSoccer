
### 迷你懂 android足球资讯应用

采用MVP模式开发，Retrofit请求数据、Glide加载图片、greenDAO保存数据到数据库
应用提供中超、西甲、英超、意甲、德甲联赛资讯和各种数据排名，球员球队基本信息，instagram与twitter实时动态
